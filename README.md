# Identification of ethical considerations in clincial trial among LMICs using LDA/NLP technique
This project applies Latent Dirichlet Allocation (LDA) topic modeling to analyze ethical concerns discussed in clinical trial publications conducted in low- and middle-income countries (LMICs).The dataset consists of bibliographic records exported from Scopus, covering publications from 2022–2025.Using natural language processing techniques, this study identifies dominant ethical themes discussed in the literature (snipshot in Result).

# Feature of this repository
	•	literature mining from Scopus
	•	NLP preprocessing pipeline
	•	topic modeling using Gensim
	•	ethical theme discovery
	•	visualization of topic structures

# Topic Modeling
Topic modeling was implemented using Gensim, a Python library for topic modeling and document similarity analysis.

The model uses Latent Dirichlet Allocation, a probabilistic model that identifies latent thematic structures in a corpus of documents.

Each document is represented as a mixture of topics, and each topic is represented by a distribution of words.
